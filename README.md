# Financial Analysis – Dadelo S.A.

## Overview
This project analyzes the impact of an Initial Public Offering (IPO) on the financial performance of Dadelo S.A., which debuted on the Warsaw Stock Exchange in December 2020.

## Scope
- Ratio analysis (profitability, liquidity, leverage, efficiency)
- Comparison of financial results before and after IPO
- Analysis period: 2017–2023
- Evaluation of short-term vs long-term impact

## Tools
- Excel
- Financial statements analysis

## Key insights
- IPO led to a short-term improvement in financial performance
- In subsequent years, financial indicators deteriorated
- Results suggest that IPO benefits were temporary

## Data sources
- Annual financial statements
- Warsaw Stock Exchange data
- Company reports
